Linux命令

Linux命令

ls

查看当前文件夹的所有文件

history

查看之前输入了哪些命令

！+输入对应的编号 就可以执行当时的命令

Linux命令结构

命令体： ls

选项： -l(详细信息)

路径： snap

pwd

查看当前路径

whoami

查看当前的root用户（我是谁）

cd

前面没有 / 代表的是相对路径

前面有 / 代表的是绝对路径

cd .. 返回上一级目录

cd ../.. 返回上上一级目录

cd / 直接返回根目录

cd ~ 回到当前用户目录

cd - 返回上一次所在的目录

常用快捷键

ctrl + c 取消当前操作

ctrl + L 清屏 或者clear

ctrl + d 退出当前用户

ctrl + r 查看历史命令

[TAB]:

​ 1、命令补全

​ 2、路径补全

ctrl + a 把光标移动到行首

ctrl + e 把光标移动到行尾

ctrl + u 把光标到行首的内容删除

ctrl + y 粘贴

ctrl + k 把光标到行尾的内容删除

ctrl + s 锁屏

ctrl + q 解屏

ctrl + > 向右移动一个单词

ctrl + < 向左移动一个单词

poweroff 关机（远程关机的话，前面要加一个sudo，然后再输入一次密码）

reboot 重启

文件操作

创建目录

相对路径是从当前开始创建的

mkdir 文件夹名

mkdir -p b/c/d 多级目录

批量创建目录

mkdir a{1..10} 或者 mkdir dir{a..f}

安装tree

sudo apt install tree

tree 查看当前

tree -L 1/ 查看当前目录下的一层

tree -L 2/ 查看当前目录下的二层

指定查看目录

ps：这里的b代指文件

相对路径

​ tree b

绝对路径

​ tree /home/b

模糊查询

*通配符

ls -ld dir* 查看所有dir同名的文件

删除文件（危险）

rm 文件名 删除文件

rmdir 文件夹名 删除文件夹 （无法删除多级目录）

rm -rf 目录 删除多级目录

修改目录

复制

​ cp 源文件 目标文件 到文件夹 复制文件（可以重新命名）

​ 案例：

​ cp aaa.py py_case/eee.py

​ aaa.py 源文件

​ py_case 目标文件夹

​ eee.py 重新命名的文件

​

​ 如果是文件夹要加 cp -r 源文件夹 目标文件夹 复制文件（可以重命名）

剪切

​ mv 源文件/源文件夹 目标文件/目标文件夹 剪切文件（可以重新命名）

普通文件

创建文件

​ touch 文件名

查看文件

​ cat 文件名

查看文件属性

​ ls -l 文件名

​ ls -ld 文件夹名

搜索文件

​ find -name ‘文件名’

​ 通配符*

​ find -name ‘*加一部分文件的前缀或者后缀’

软硬链接（数据发生变化会同步）

软连接 快捷方式

​ ln -s 源文件 软链接文件

影链接 相当于复制 备份

​ ln 源文件 硬链接文件

vim操作

修改文件内容（重点）

vim模式：命令模式（光标，复制），插入模式（编辑），末行模式（保存退出）

进入 vim 文件名

切换模式 esc键

​ ：选择末行模式（shift + ：）

​ w 保存

​ q 退出

进入插入模式 切换模式按esc键

​ i 光标的前一个位置插入

​ a 光标的后一个位置插入

​ I （可以按shift + i）跳到行首

​ A （可以按shift + a）跳到行尾

​ o 当前光标所在行下方新增一行

​ O 当前光标所在行上方新增一行

命令模式

​ gg 跳到第一行行首

​ G 跳到最后一行行首

​ ^ 跳到非空格行首

​ $ 跳到本行行首

操作文件

​ yy 复制整行内容

​ p 粘贴

​ dd 删除当前行

运行python代码（vim模式下）

​ 1、f5 运行代码 回车重新进来

​ 2、也可以退出保存文件vim python 文件名.py 运行代码

进程

ps 是操作系统的进程命令

查看所有的进程

​ ps -ef

选择查看进程

​ ps -ef |gref 想要的进程

关闭进程

​ kill 进程的id

​ pkill 进程名字
